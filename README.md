# Electro - Full Stack E-Commerce Website (PHP + MySQL)

Electro is a modern full-featured E-Commerce website built using PHP, MySQL, TailwindCSS, and Bootstrap. It includes a complete user system, product management, shopping cart, order tracking, and admin dashboard.

---

## Features

### User Features
- User registration and login
- Browse products by category
- View product details
- Add to cart
- Checkout with Razorpay payment integration
- Order tracking with dynamic timeline
- View order history
- Write product reviews with star ratings

---

### Admin Features
- Admin dashboard with analytics
- Product management (Add, Edit, Delete)
- Category management
- Order management and status update
- User management (block / unblock / delete)
- Sales chart with real data

---

### UI Features
- Modern responsive UI
- Material Dashboard admin panel
- Amazon-style product page
- Dynamic order timeline
- Clean TailwindCSS design

---

## Tech Stack

Frontend:
- HTML5
- TailwindCSS
- Bootstrap
- JavaScript
- Chart.js

Backend:
- PHP (Core PHP)
- MySQL

Payment Gateway:
- Razorpay
